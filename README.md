Steps to run locally

1. Add in vm options
   -Djava.awt.headless=false

2. Create a secrets.properties and add your gemini key 
   API_KEY = .........................

3. Add tessaract datapath as well as install the tessaract based on your system configurations
   tesseract.datapath = C:/Program Files/Tesseract-OCR/tessdata (Depends where you installed the binaries)

4. Try prompts like "Go to chrome search for bajaj stock then go to 'News' section then click on all links","Open chrome search for wordle".


I have performed the following actions:

1. Searched for 'bajaj stock' in Chrome.
2. Clicked on 'News'.
3. Ctrl-clicked all blue hyperlinks found on the screen, which were 6 in total.
